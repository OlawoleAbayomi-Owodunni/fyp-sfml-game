FloorPlan FloorGenerator::generateFloorPlan ( int floorId,
int dungeonSeed,
bool isBossFloor )

Generates a floor plan describing room types and their connectivity graph.

The plan is generated deterministically from dungeonSeed and floorId. The resulting graph ensures:

  • Room 0 is always the spawn room.
  • The last room is always the portal room (boss room placeholder on final floor).
  • The spawn and portal rooms are not directly connected by an edge.
  • Each room is connected to at least one earlier room (connected graph).
