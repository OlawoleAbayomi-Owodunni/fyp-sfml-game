Constructs a timed damage trigger.

The trigger uses CollisionLayer::DAMAGE_TRIGGER_LAYER and a caller-provided mask to control which entities can be affected.

Parameters
centerWorld-space center of the trigger.
sizeWorld-space size of the trigger.
lifetimeLifetime in seconds before auto-destruction.
damageDamage dealt when overlapping.
targetMaskCollision mask describing valid damage targets.
